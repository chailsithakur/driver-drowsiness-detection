# Driver Drowsiness Detection System

## Overview
This project detects driver drowsiness using Computer Vision techniques. It monitors the driver's eyes through a webcam and generates an alert when signs of drowsiness are detected.

## Technologies Used
- Python
- OpenCV
- dlib
- imutils
- pygame

## Features
- Face Detection
- Eye Detection
- Eye Aspect Ratio (EAR)
- Drowsiness Alert
- Real-time Webcam Monitoring
